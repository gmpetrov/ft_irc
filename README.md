ft_irc
======

School Project : IRC Client & Server

To launch :

make

./serveur <port>

./client 127.0.0.1 <port>

Implemented commands :

    change nickname : /nick <nickname>
    
    join channel    : /join <#chan>
    
    leave channel   : /leave <#chan>
    
    private msg     : /msg  <nickname> <msg>
    
    list people
    connected on a
    channel         : /who
